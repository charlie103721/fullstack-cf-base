# M2 — About Page

## Phase 1: Create about page

- Create a new route at `/about` in the React client
- Add content:
  - Page heading: "About"
  - Brief description of the app
  - A "Back to Home" link that goes to `/`
- Style using Tailwind CSS
- Create component in `client/pages/About.tsx`
- Add the route to the React Router config in `client/App.tsx`
